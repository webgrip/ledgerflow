# Demo Script

Use this script for interview prep or project walkthroughs.

## 30-day demo

1. Open the app.
2. Log in as demo user.
3. Show organization/workspace.
4. Create or view accounts.
5. Create a transaction.
6. Show account activity.
7. Show balance.
8. Ask AI to explain a transaction.
9. Show tests passing.
10. Explain architecture and roadmap.

## 60-day demo

1. Show webhook ingestion.
2. Process a provider event.
3. Replay duplicate webhook and show it is ignored.
4. Show queued processing.
5. Run reconciliation.
6. Show reconciliation issue.
7. Ask AI to explain the mismatch.
8. Show Horizon/Pulse.

## 90-day demo

1. Show operations dashboard.
2. Show audit log.
3. Show export flow.
4. Show AI analysis history.
5. Show read-only MCP tool.
6. Show security/authorization tests.
7. Explain production deployment plan.

## Interview talking points

- Laravel-first modular monolith
- fintech domain constraints
- idempotent external event handling
- AI as advisory layer
- Boost for development workflow
- tests and CI
- observability and operations
